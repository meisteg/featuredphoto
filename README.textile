h1. Featured Photo for phpWebSite

"Featured Photo":http://github.com/meisteg/featuredphoto is a module for the "phpWebSite":http://phpwebsite.appstate.edu CMS that lets you add photo blocks to your website. The photos can be hosted locally or on "Flickr":http://www.flickr.com.

h2. Getting Started

* "Installation Instructions":http://wiki.github.com/meisteg/featuredphoto/install for this module.
* "Submit an issue":http://github.com/meisteg/featuredphoto/issues if you have questions or to report a problem.
